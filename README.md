# 📋 تطبيق "مهامي" - إدارة المهام

تطبيق ويب بسيط ومتجاوب لإدارة المهام اليومية، مع دعم مستويات الأولوية والفلترة، يعمل بالكامل في المتصفح ويعتمد على **Local Storage** لحفظ البيانات.

## ✨ الميزات

- ✅ **إضافة مهام متعددة** دفعة واحدة (كل مهمة في سطر جديد)
- 🎯 **ثلاثة مستويات للأولوية**: ضروري 🔴، مهم 🟡، عادي 🟢
- 🔍 **فلترة المهام** حسب الأولوية
- 📊 **إحصائيات فورية** (إجمالي، منتهية، متبقية)
- ✏️ **تعديل وحذف** المهام بسهولة
- 🔄 **استرجاع المهام** المنتهية
- 💾 **حفظ تلقائي** في Local Storage
- 📱 **تصميم متجاوب** مع الهواتف والأجهزة اللوحية
- 🎨 **واجهة مستخدم جذابة** مع حركات وأنيميشن

## 🚀 كيفية الاستخدام

### 1️⃣ صفحة البداية (Splash)
تعرض إحصائيات سريعة عن مهامك:
- إجمالي المهام
- المهام المنتهية
- المهام غير المنتهية

### 2️⃣ إضافة المهام
- انتقل إلى تبويب **"إضافة"**
- اكتب كل مهمة في سطر منفصل
- اختر مستوى الأهمية الافتراضي
- اضغط **"حفظ جميع المهام"**

### 3️⃣ عرض المهام
- انتقل إلى تبويب **"المهام"**
- عرض جميع المهام غير المنتهية
- يمكنك:
  - ✅ تحديد المهمة كمنتهية
  - ✏️ تعديل النص
  - 🗑️ حذف المهمة
  - 🎯 تغيير مستوى الأهمية
- فلترة المهام باستخدام الأزرار السفلية

### 4️⃣ المهام المنتهية
- انتقل إلى تبويب **"منتهية"**
- عرض المهام المنجزة
- استرجاع المهمة إلى القائمة الرئيسية

### 5️⃣ العودة للصفحة الرئيسية
- اضغط على زر **"الرئيسية"** في أعلى التطبيق
